# AkademiX Dashboard

> Türk-Alman Üniversitesi öğrencileri için akademik yönetim platformu

---

## 🎓 Nedir?

AkademiX, TAU öğrencilerinin akademik hayatını tek bir yerden yönetmesini sağlayan bir web uygulamasıdır. Google Classroom entegrasyonu sayesinde ödevler, duyurular ve ders materyalleri anlık olarak takip edilebilir.

## ✨ Özellikler

- **📅 Haftalık Ders Programı** — 219 dersin renk kodlu haftalık görünümü
- **📚 Derslerim** — Kayıtlı derslerin filtrelenebilir kart görünümü
- **🔗 Google Classroom** — Ödevler, duyurular ve notlar anlık senkronizasyon
- **📁 Materyaller** — Dosya yükleme, bağlantı ve not ekleme
- **🤖 AI Özetleme** — PDF ve belgelerin yapay zeka ile otomatik özetlenmesi
- **📝 Not Defteri** — Derse özel not alma ve düzenleme
- **📆 Takvim** — Etkinlik ve ödev takibi
- **🌙 Light / Dark Tema** — Göz dostu tema geçişi

## 🚀 Kullanım

1. [akademix.tau.edu.tr](https://lefos5.github.io/akademix) adresine git
2. **TAU kurumsal Google hesabınla** giriş yap (`@tau.edu.tr`)
3. Derslerini seç ve kullanmaya başla

> ⚠️ Bu uygulama yalnızca TAU kurumsal Google hesaplarıyla kullanılabilir.

## 🛠️ Teknolojiler

- Vanilla HTML / CSS / JavaScript
- Google Classroom API v1
- Google OAuth 2.0
- Claude AI API (Anthropic)
- PDF.js — PDF metin çıkarma
- Mammoth.js — Word dosyası desteği
- IndexedDB — Dosya depolama
